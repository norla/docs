Yaml aint markup language

Data serialization language designed to be human-friendly.

Sequences (lists)
- - - - - - - - - -
Each item is prefixed with whitespace and a dash (" - ")
```
 - a
 - b
 - c
```


Mappings (key/val)
- - - - - - - - - -
Use colon

```
key: 1
animals:
 - Dog
 - Cat
```
